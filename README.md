# Chatwork マイタスク リマインダー

Chatwork の未完了タスクを定期的に取得し、スプレッドシートへの記録と Bot メッセージによる通知を自動化する Google Apps Script ツールです。

---

## 機能概要

- Chatwork API からログインユーザーの未完了タスクを取得
- 取得したタスクをスプレッドシート（`my_tasks` シート）に書き込み
- Chatwork Bot 経由で指定ルームへリマインドメッセージを送信
- 時刻トリガーを最大 2 件登録し、毎日指定時刻に自動実行

---

## 事前準備

### 1. Chatwork API トークンの取得

| トークン種別     | 用途                           | 取得場所              |
| ---------------- | ------------------------------ | --------------------- |
| `CHATWORK_TOKEN` | タスク一覧取得（個人トークン） | Chatwork 設定 → API   |
| `BOT_TOKEN`      | Bot によるメッセージ送信       | Chatwork Bot 管理画面 |

### 2. ライブラリの追加

GAS エディタの「ライブラリ」から `ChatWorkClient` を追加します。

| 項目          | 値                                                          |
| ------------- | ----------------------------------------------------------- |
| スクリプト ID | `1nf253qsOnZ-RBd7fFIBMQFb7MFMxKQMzEjcWIZwkGeTak7qoVCTRoXdL` |
| バージョン    | 最新の安定版                                                |

### 3. スクリプトプロパティの設定

GAS エディタの「プロジェクトの設定」→「スクリプトプロパティ」に以下を登録します。

| キー             | 値                |
| ---------------- | ----------------- |
| `CHATWORK_TOKEN` | 個人 API トークン |
| `ROOM_ID`        | 通知先ルーム ID   |
| `BOT_TOKEN`      | Bot API トークン  |

> スクリプトプロパティが未設定の場合は `Config` オブジェクトの値にフォールバックします。

### 4. スプレッドシートの準備

`my_tasks` という名前のシートを作成し、以下のセルに値を設定します。

| セル | 内容                                       |
| ---- | ------------------------------------------ |
| B1   | 通知先メンバーの Chatwork アカウント ID    |
| C1   | 通知先メンバーの表示名                     |
| B4   | 1 回目のリマインド時刻（時・数値）例: `9`  |
| C4   | 2 回目のリマインド時刻（時・数値）例: `18` |

タスクデータは 6 行目以降に自動書き込みされます。

| 列   | 内容               |
| ---- | ------------------ |
| A 列 | 期限               |
| B 列 | グループチャット名 |
| C 列 | タスク依頼者       |
| D 列 | タスク詳細         |

## 関数リファレンス

### `alert_my_task()`

メイン処理。Chatwork からタスクを取得し、スプレッドシートへの書き込みと Bot 通知を実行します。トリガーからの自動実行、または手動実行の両方に対応します。

### `set_trigger()`

B4 セルの時刻を読み取り、`alert_my_task` を毎日指定時刻に実行するタイムトリガーを作成します。生成されたトリガー ID は D4 セルに保存されます。

### `set_trigger_second()`

C4 セルの時刻を読み取り、2 つ目のタイムトリガーを作成します。トリガー ID は E4 セルに保存されます。

## 使い方

### 初回セットアップ

1. スプレッドシートの各セルに値を設定する（[スプレッドシートの準備](#4-スプレッドシートの準備) 参照）
2. GAS エディタから `set_trigger` を実行 → 1 回目のリマインド時刻を登録
3. 必要に応じて `set_trigger_second` を実行 → 2 回目のリマインド時刻を登録
4. 動作確認として `alert_my_task` を手動実行

### 手動実行

GAS エディタまたはスプレッドシートのメニューから `alert_my_task` を直接実行できます。

### トリガーの変更・削除

GAS エディタの「トリガー」画面から既存トリガーを削除後、再度 `set_trigger` / `set_trigger_second` を実行してください。

## 注意事項

- Chatwork API の利用制限（レートリミット）に注意してください。頻度の高いトリガー設定は避けてください。
- `BOT_TOKEN` はメッセージ送信専用です。タスク取得には個人の `CHATWORK_TOKEN` が必要です。
- タスク件数が 0 件の場合は「未完了タスクはございませんでした。」と通知してスプレッドシートへの書き込みはスキップされます。
